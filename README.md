# CN-La-Carolina-Gestión-de-Trofeos-de-Natación
Aplicación web de página única para gestionar trofeos y competiciones de natación organizados por el Club de Natación La Carolina. Permite administrar clubes, nadadores e inscripciones, generar carreras automáticamente respetando categorías y calles, tomar tiempos y producir clasificaciones con medallas, además de imprimir hojas de carrera y resultados finales.
Características principales

Gestión de varias competiciones con sus participantes, clubes y resultados
14 categorías oficiales (Prebenjamín a Máster, femenino y masculino)
Generación automática de carreras con asignación de calles centradas y preferencia para el club anfitrión
Toma de tiempos con validación en tiempo real y autoguardado
Carreras de relevos por clubes
Clasificaciones por categoría y estilo con medallas oro/plata/bronce
Impresión PDF de hojas de carrera (con o sin tiempos) y clasificación completa
Importación/exportación de datos en JSON y CSV
Almacenamiento local en el navegador (sin servidor)

Tecnología
HTML + CSS + JavaScript puro, sin dependencias. Funciona abriendo index.html directamente en el navegador. Datos persistidos en localStorage.
